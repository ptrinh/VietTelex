Tiếng Việt

Type Vietnamese on your Mac,
fast, smooth, right for 2026

VietTelex (ViệtTelex) is a free, open-source Telex input method for macOS, built on Apple's InputMethodKit for deep integration — no underline under the word you're typing, the caret always stays at the end, and it works in Terminal and Chrome too.

 
⬇︎ Download latest (.pkg)

Free · Open source (MIT) · Notarized by Apple · macOS 14+

Why VietTelex?

Rewritten from scratch after nearly 20 years on macOS without a Vietnamese input method that felt truly good, correct and smooth.

MinimalistTelex only, precomposed Unicode only — no extra features.
IntuitiveBehaves exactly like a system input method — nothing to relearn.
Performant~130 ns/key, no background process — smooth no matter how fast you type.

● Native to Apple's IMKit

A real system input method, not a key-blocking app — so per-app typing modes, macOS spelling/prediction/undo/VoiceOver all work correctly.

● Fast — ~130 nanoseconds/key

Spelling rules compiled into a state machine, zero allocation while typing. Type as fast as you like — no lost or scrambled characters.

● Works in Terminal & Chrome

Terminal/iTerm keep the shell's autocomplete; Chrome's address bar, Excel cells and Spotlight all type correctly — where other input methods often break.

● Smart with English & code

Auto-restores google, github…; recognizes camelCase tokens (OmS, JavaScript) so it won't add tone marks by mistake; strict Telex mode for devs.

● Light & private

No background CPU drain, no network connections, collects no data whatsoever.

Install in a minute

The installer places the app, registers the input method and opens System Settings for you — no drag-and-drop, no logout.

Download and run the installer

Download VietTelex-x.y.z.pkg (signed + notarized by Apple) and double-click. Or use Homebrew: brew install --cask ptrinh/viettelex/viettelex

Add the input method

System Settings → Keyboard → Input Sources → Edit… / ＋ → Vietnamese → ViệtTelex → Add.

Give it a try

Switch to ViệtTelex with the 🌐 key or ⌃Space. To type in Terminal/iTerm/Chrome, grant Accessibility to VietTelex (Privacy & Security → Accessibility).
